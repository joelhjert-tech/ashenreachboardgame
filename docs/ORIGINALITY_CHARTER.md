# Originality Charter

Ashen Reach is an original setting, ruleset, and user interface. This project
must avoid borrowed names, lifted mechanics text, derivative lore, copied card
copy, or visual homage that would make it read like an adaptation of an
existing tabletop property.

The practical rule is simple: if a term, faction identity, card concept, stat
label, board region, or story element is recognizable as belonging to another
franchise, it does not belong in this repository. Ashen Reach should instead
use its own vocabulary, its own thematic logic, and its own structure for
adventure play.

Every future addition should be checked against that standard before it lands:

- Original names and terminology only
- Original setting lore and artifact concepts only
- Original board geography and progression only
- Original UI language and iconography only
- No copied or paraphrased text from protected source material
